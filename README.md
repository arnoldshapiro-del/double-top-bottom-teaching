# Trading Double Bottoms and Double Tops, and Bull Flags and Bear Flags

M2K & MES 2-minute scalping reference with inline plain-English teaching explanations for all four patterns.

## What's in here

- `index.html` — the full single-file app (Teaching Edition, v2.0)
- `netlify.toml` — Netlify build/deploy config
- Pure HTML + CSS + vanilla JS. No build step. No dependencies beyond Google Fonts (loaded at runtime).

## Features

- **Four pattern tabs** — Double Bottom, Double Top, Bull Flag, Bear Flag, plus a Unified Principles reference tab
- **Reversal patterns** (Double Top / Double Bottom) with quality scoring (0–5), red-flags panel, and the final "Four Principles" discipline gate
- **Continuation patterns** (Bull Flag / Bear Flag) with precision definitions, four entry methods each, three stop-loss methods each, M2K vs MES comparison, and Arnie's personal approach workflows
- **Unified Principles** — an 8-point cross-pattern reference covering confirmation, risk, patience, trend alignment, measured moves, volume, higher-timeframe filters, and emotional discipline
- Trade journal with localStorage persistence and CSV export
- Print-friendly layout (landscape, Letter) — print view shows all four pattern panels sequentially with page breaks
- Plain-English teaching explanations under every concept

## Local development

Just open `index.html` in a browser. No server required. For hot-reload during development, use the included `serve-dtb-teaching.js` static server on port 3478.

## Deployment

Connected to Netlify via GitHub auto-deploy. Any push to `master` triggers a rebuild.

Live site: https://arnie-double-top-bottom.netlify.app
