# Double Top / Double Bottom — Teaching Edition

M2K 2-minute scalping reference with inline plain-English explanations for every concept.

## What's in here

- `index.html` — the full single-file app (teaching edition, v1.1)
- `netlify.toml` — Netlify build/deploy config
- Pure HTML + CSS + vanilla JS. No build step. No dependencies beyond Google Fonts (loaded at runtime).

## Features

- Double Top + Double Bottom checklists with quality scoring (0–5)
- Red-flags panel, final "Four Principles" discipline gate
- Trade journal with localStorage persistence and CSV export
- Print-friendly layout (landscape, Letter)
- Plain-English teaching explanations under every concept

## Local development

Just open `index.html` in a browser. No server required.

## Deployment

Connected to Netlify via GitHub auto-deploy. Any push to `main` triggers a rebuild.
